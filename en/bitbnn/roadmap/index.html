協力募集 issue​

  • BotUser の作成 1人1日30個の取引 実装は協力スタッフに依頼
  1. ボットは ABC グループに分かれている A1-C3 まで 9人
  2. MongoDB に BotUser の情報を入れておく
  3. bittreebotDB botusers スタッフ専用のAPIキーを発行
  4. サーバー言語は自由　特に希望がなければ TypeScript on Deno
  5. でははじめにボットのプロフィールをみるAPIを先に作る
  6. ボットの情報をDBから取得 所持金とポケット addr のみで見れる ログで確認
  7. 既存APIを説明 残高参照,ビットフルーツ,ポケット,Tx
  8. 作るメソッドは run_bot_users(group: String) -> String
  9. 引数はグループ 戻り値は リザルト 例: "X" -> "対応していないグループです"
  10. 該当グループのボット1人につき30個取引してもらう 赤字OK
  11. プロフィールを見てボットがちゃんと動いたかどうか確認
  12. Go, Python が良いときは Heroku, API実装を軽く調査 flask 程度でOK
  13. URL は run-bot-users/:group メソッドは GET レスポンスは { "result": result }
  14. GitHubとDenoDeployをCD連携 自動デプロイできることを version で確認
  15. 最後に 0, 8, 16 時に A, B, C を呼び出す cron の設定をして完了
